# Планировщик полётов для MSFS 2020 (Telegram-бот)

Бот подбирает активную ВПП по ветру, автоматически выбирает процедуры
**SID / STAR / заход**, прокладывает **маршрут по воздушным трассам** в формате
ИКАО, подбирает **запасной аэродром** при непогоде и показывает погоду
(METAR/TAF) для пары аэропортов.

- **Навданные:** Navigraph AIRAC (native DFD), мировое покрытие.
- **Погода:** [aviationweather.gov](https://aviationweather.gov/data/api/) — бесплатно, без ключа.
- **Стек:** Python 3.14, aiogram 3.x.

## Установка

```bash
cd flightplanner_bot
python -m venv .venv
.venv\Scripts\activate            # Windows
pip install -r requirements.txt
```

## 1. Импорт навбазы (разово, при смене AIRAC)

```bash
python -m navdata.importer
# или указать архив явно:
python -m navdata.importer "C:\путь\navdata_native_XXXX.zip"
```
Создаёт `data/navdata.sqlite`. Путь к архиву по умолчанию задан в `config.py`.

## 2. Токен бота

Скопируйте `.env.example` в `.env` и впишите токен от [@BotFather](https://t.me/BotFather):
```
BOT_TOKEN=123456:ABC...
```

## 3. Запуск

```bash
python main.py
```

## Команды бота

| Команда | Описание |
|---|---|
| `/plan` | Пошагово: вылет → назначение → план (ВПП + SID/STAR/заход) |
| `/metar <ICAO>` | Текущая погода + построчная расшифровка METAR-кода |
| `/taf <ICAO>` | Прогноз TAF |
| `/rwy <ICAO>` | Активная ВПП по ветру (с альтернативами) |
| `/cancel` | Отмена ввода |

## Логика подбора

- **Активная ВПП** — максимум встречной составляющей ветра
  (`HW = V·cos(курс − направление ветра)`); при штиле — самая длинная полоса.
  Посадочная ВПП выбирается среди полос с опубликованным заходом (если
  оптимальная по ветру полоса без захода — берётся лучшая с заходом, с пометкой).
- **SID** — из процедур активной ВПП вылета берётся та, чья выходная точка
  ближе всего по направлению к пункту назначения.
- **STAR** — процедура, чья точка входа направлена в сторону аэропорта вылета.
- **Заход** — по активной ВПП посадки, приоритет точности: ILS → GLS → RNAV → LOC → VOR → NDB.
- **Маршрут** — A*-поиск по графу воздушных трасс от точки выхода SID до точки
  входа STAR; при разрывах трасс — короткие DCT-перемычки. Вывод в формате ИКАО:
  `DEP SID FIX AIRWAY FIX … STAR DEST`.
- **Запасной аэродром** — при непогоде (IFR/LIFR, сильный ветер) ближайший
  аэропорт с ВПП ≥ 6500 фт и приемлемой погодой (категория VFR/MVFR).

## Структура

```
flightplanner_bot/
├── config.py             # пути и параметры
├── main.py               # запуск бота
├── navdata/
│   ├── importer.py       # Navigraph → SQLite
│   └── db.py             # доступ к навбазе
├── core/
│   ├── weather.py        # METAR/TAF
│   ├── runway.py         # выбор активной ВПП
│   ├── procedures.py     # автоподбор SID/STAR/Approach
│   ├── router.py         # маршрутизация по воздушным трассам (A*)
│   ├── alternate.py      # подбор запасного аэродрома
│   └── planner.py        # сборка карточки плана
└── bot/handlers.py       # хендлеры aiogram + FSM
```

## Примечания

- Направление ветра METAR — от истинного севера, курс ВПП — магнитный;
  для выбора полосы склонением пренебрегаем (стандартно для симуляторов).
- Данные предназначены только для использования в симуляторе, не для реальной навигации.
